# 全部账号汇总收口说明

`ACCOUNT-ALL-001` 当前不重新开放跨账号分析视图。原因是不同账号之间的保底、时间线、资源、平均出货和分享口径没有统一契约，直接合并会产生误导性指标。

## 当前保留

- 同一账号内的“全部卡池总览”继续保留，用于跨卡池时间线回顾。
- 导出弹窗中的“所有账号记录”继续保留，用于显式导出原始记录。
- 没有选中账号时，分析页回退到一个有效账号，不再隐式合并多个账号。

## 当前关闭

- 未选账号时的跨账号 merged analysis。
- 多账号合并后的当前保底、软保底概率、平均 UP 和资源口径。
- 把“全部账号”作为默认分析状态展示给用户。

## 重新开放条件

重新开放跨账号汇总前，必须先定义并测试以下契约：

- 哪些指标可以聚合：总抽数、按卡池计数、按稀有度计数。
- 哪些指标必须隐藏或改名：当前保底、软保底概率、资源余额、抽卡计划。
- 时间线如何展示账号来源、排序和重复记录。
- 分享图、导出、桌面端和移动端是否使用同一套文案与字段。
- 旧导出和 BOT 文案不能把原始记录导出误称为分析汇总。
